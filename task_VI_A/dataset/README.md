
# Task VI.A - Super Resolution (Simulated Data)

This task focuses on training a deep learning-based super-resolution model using simulated gravitational lensing images.

## Objectives
- Train a model to upscale low-resolution (LR) images to high-resolution (HR)
- Implement using PyTorch or TensorFlow
- Evaluate using:
  - MSE
  - PSNR
  - SSIM
