# Common Test I - Multi-Class Classification

This task involves building a deep learning model to classify strong gravitational lensing images into three categories:

- No Substructure
- Subhalo Substructure
- Vortex Substructure

## Objectives
- Train a classification model using PyTorch or TensorFlow
- Perform a 90:10 train-test split
- Evaluate performance using ROC Curve and AUC Score
